# 算24点 (24 Point Game)

一个基于 Rust 和 Web 的算24点游戏程序。

## 功能特性

- 🚀 Rust 命令行程序
- 🌐 本地 Web 服务器（端口：8088）
- 🎨 现代化的渐变色界面设计
- 🔄 自动打开浏览器

## 快速开始

### 安装依赖

确保已安装 Rust 工具链。如果没有，请访问 [https://rustup.rs/](https://rustup.rs/) 安装。

### 运行程序

```bash
cargo run
```

程序将：
1. 启动本地 Web 服务器在 `http://localhost:8088`
2. 自动在浏览器中打开页面
3. 显示"算24点"标题页面

### 停止服务器

按 `Ctrl+C` 停止服务器。

# 需求概览

## 项目背景

算24点游戏程序，基于 Rust 后端和 React 前端实现。

## 功能需求

### 已上线功能

| 功能 | 描述 | 状态 |
|------|------|------|
| 计算24点解法 | 输入4个数字，返回所有可能的解法 | ✅ 已上线 |
| 自动出题 | 生成一个有解的24点题目 | ✅ 已上线 |
| 支持扑克牌数字 | A=1, J=11, Q=12, K=13 | ✅ 已上线 |
| 基本运算支持 | 加减乘除 | ✅ 已上线 |
| 高级运算支持 | 阶乘、乘方、开方、对数 | ✅ 已上线 |
| 现代化UI | 渐变色界面设计 | ✅ 已上线 |

## 进行中需求

| 功能 | 描述 | 状态 |
|------|------|------|
| 扑克牌背景装饰 | 紫色背景上添加半透明扑克牌图案 | ✅ 已完成 |

## 非功能需求

- 高性能计算（Rust 后端）
- 现代化响应式前端
- 支持生产模式静态文件服务
- 支持开发模式 API 服务

## 项目知识

### 算24点游戏规则
- 给定4个数字，通过加减乘除等运算得到24
- 支持扑克牌表示：A=1, J=11, Q=12, K=13
- 每个数字必须使用且只能使用一次

### 算法知识
- 递归回溯求解所有可能解法
- 浮点数比较使用 epsilon (1e-6)
- 阶乘限制在 0-12 避免数值溢出
- 开方和对数运算有输入限制
